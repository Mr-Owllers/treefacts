TreeFacts

A CLI application that displays facts about trees

How to download the TreeFacts

Requirements-

  • GCC
  • CURL

Unix-Like

git clone https://github.com/Mr-Owllers/treefacts.git
cd treefacts
make install

Windows
!! Warning: This app has not been tested on Windows devices.

git clone https://github.com/Mr-Owllers/treefacts.git
cd treefacts
gcc src/treefacts.c -o obj/treefacts.o
gcc src/libtreefacts.c -o obj/libtreefacts.o
gcc obj/treefacts.o -o obj/treefacts.exe

More information about the app can be found in the git repo
